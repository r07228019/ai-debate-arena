  • 「民生 vs 國防」可能是個假議題：真正的問題不是預算怎麼分，而是你願意為什麼樣的生活方式付代價。迴避這題，吵數字只是轉移焦點。
  • 「買」和「會買」是兩回事：支持增加國防預算，不等於支持現行採購案的每一項；監督採購黑箱與反對軍購是兩種完全不同的立場，別被政治口水混為一談。
  • 和平有兩種：一種是有實力撐腰的和平，一種是沒實力之後的「安靜」。霍布斯那句「沒有公權力保障的和平叫臣服」值得每個談避戰的人先想一遍，再決定自己真正要的是哪一種。
